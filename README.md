# Kyle Karpack's Portfolio

The professional portfolio of Kyle Karpack

**Technology:**

- [Remix][remix]
- [React](https://reactjs.org)
- [TailwindCSS](https://tailwindcss.com)
- [GraphCMS][graphcms]
- [Netlify][netlify]

## Setup

```bash
# Add the following variables to your .env file
<GRAPHCMS_ADMIN>
<GRAPHCMS_TOKEN>
<GRAPHCMS_URL>

# Install dependencies
npm install

# Start building
npm run dev
```
